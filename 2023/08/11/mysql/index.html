1.8 部分常见函数

a. 函数详解
函数 含义
ROUND(x,y) 四舍五入函数（y为负数时，保留小数点左边相应位数为0）
CONCAT(s1,s2,…) 连接字符串函数
REPLACE(s,s1,s2) 使用s2替代s中的所有的s1
LEFT(s,n) LEFT(s,n) substring(s,n,len) 截取最左边n的字符串 最右边n个字符串 字符串s中从第n个字符起len个长度字符串（n可以为负数，即从到处第n个字符取）
CAST(x as type) 转换数据类型，type可填char(n)、date、time、datetime、decimal
year(date)、month(date)、day(date) 获取日期中的年月日
date_add(date,interval expr type) date_sub(date,interval expr type) 指定时间进行加减。如date_add(‘2023-.08-08 22:10:03’,interval 1 second)返回 ‘2023-.08-08 22:10:04’
datediff(date1,date2) 计算两个日期之间间隔天数
date_format(date,format) 将日期格式化
IF(expr,v1,v2) 条件判断
CASE WHEN 用法1：case 表达式 when v1 then r1 [when v2 then r2 …] end / 用法2：case when v1 then r1 [when v2 then r2 …] end
